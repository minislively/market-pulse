# Vision

`market-pulse` exists to turn terminal wait time into market reasoning practice.

The user context is simple: while AI/coding tasks run, it is easy to drift into phone scrolling. `market-pulse` keeps the loop in the terminal: see a short market card, write one interpretation, receive counter-view feedback, and review the accumulated reasoning later.

## North star

Market literacy through repeated, low-friction, terminal-native reflection.

## Core loop

```text
collect context -> show pulse -> user writes thought -> feedback -> journal -> review
```

## Boundaries

`market-pulse` should not become:

- a trading bot
- a buy/sell recommender
- a portfolio tracker
- a charting-first terminal
- a backtesting platform

Those may be adjacent tools, but this project is a market thinking journal.
